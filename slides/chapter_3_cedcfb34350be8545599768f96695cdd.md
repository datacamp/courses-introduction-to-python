`@script`
그러기 전에, 패키지 전체나 패키지의 특정 모듈을 import로 가져와야 합니다. import 문을 사용하면 됩니다.

numpy 패키지 전체를 가져오려면 이렇게 import numpy라고 하면 됩니다.

NumPy에서 자주 쓰는 함수로 array가 있습니다. 리스트를 입력으로 받죠. 이렇게 그냥 array 함수를 호출하면 오류가 납니다.

numpy 패키지의 array 함수를 참조하려면 이렇게 해야 합니다.

이번에는 동작하죠. NumPy 배열은 데이터 사이언스에 아주 유용하지만, 자세한 내용은 나중에 다루겠습니다.

매번 numpy 점 표기를 쓰는 게 번거로울 수 있어서, 패키지를 가져온 뒤 다른 이름으로 부를 수도 있습니다. import 문에 as를 덧붙여 이렇게 하면 돼요.

이제부터는 numpy.array 대신, NumPy의 array 함수를 쓰려면 np.array를 사용하면 됩니다.

패키지에서 특정 함수 하나만 필요할 때도 있습니다. 파이썬은 이를 코드에서 명시적으로 표현할 수 있게 해줍니다. 예를 들어 NumPy에서 array 함수만 쓰고 싶다면, import numpy 대신 from numpy import array처럼 할 수 있습니다.

이번에는 이렇게 array 함수를 그냥 호출하면 됩니다. 여기서는 numpy 점 표기가 필요 없어요.

이처럼 패키지의 일부분만 from import로 가져오면 코드를 줄이는 데는 도움이 되지만, 맥락을 일부 잃게 됩니다.
