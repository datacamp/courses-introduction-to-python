`@script`
Sebelum bisa melakukan itu, Anda harus mengimpor paketnya, atau modul tertentu dari paket tersebut. Anda dapat melakukannya dengan pernyataan import.

Untuk mengimpor seluruh paket numpy, Anda bisa mengetik import numpy, seperti ini.

Salah satu fungsi yang sering dipakai di NumPy adalah array. Fungsi ini menerima daftar sebagai input. Memanggil fungsi array begitu saja seperti ini akan menimbulkan error.

Untuk merujuk ke fungsi array dari paket numpy, Anda perlu yang ini.

Kali ini berhasil. Array NumPy sangat berguna untuk sains data, tetapi nanti kita bahas lebih lanjut.

Memakai awalan numpy titik terus-menerus bisa cukup melelahkan, jadi Anda juga bisa mengimpor paketnya dan merujuknya dengan nama lain. Anda dapat melakukannya dengan menambahkan as pada pernyataan import, seperti ini.

Sekarang, alih-alih numpy.array, Anda cukup memakai np.array untuk menggunakan fungsi array milik NumPy.

Ada kalanya Anda hanya butuh satu fungsi tertentu dari sebuah paket. Python memungkinkan Anda menyatakannya secara eksplisit di kode. Misalkan kita hanya ingin memakai fungsi array dari paket NumPy. Alih-alih import numpy, Anda bisa menulis dari numpy import array, seperti ini.

Kali ini, Anda bisa cukup memanggil fungsi array seperti ini, tidak perlu memakai numpy titik di sini.

Versi dari import untuk memakai bagian-bagian spesifik dari sebuah paket bisa berguna untuk mengurangi pengetikan, tetapi Anda juga kehilangan sebagian konteks.
