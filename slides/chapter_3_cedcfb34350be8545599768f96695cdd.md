`@part2`
```py

```{{6}}
```py

```{{7}}

```out

```{{8}}

```py
from numpy import array
```{{9}}
```py
array([1, 2, 3])
```{{10}}

```out
array([1, 2, 3])
```{{11}}

`@script`
Prima di farlo, devi importare il pacchetto, o un modulo specifico del pacchetto. Puoi farlo con l’istruzione import.

Per importare l’intero pacchetto numpy, puoi scrivere import numpy, così.

Una funzione molto usata in NumPy è array. Prende in input una lista. Se la chiami semplicemente così, otterrai un errore.

Per riferirti alla funzione array del pacchetto numpy, ti serve questo.

Questa volta funziona. L’array di NumPy è molto utile per fare data science, ma ne parleremo meglio tra poco.

Usare ogni volta il prefisso numpy punto può diventare un po’ noioso, quindi puoi anche importare il pacchetto e riferirti ad esso con un nome diverso. Puoi farlo estendendo l’istruzione di import con as, così.

Ora, invece di numpy.array, userai np.array per richiamare la funzione array di NumPy.

Ci sono casi in cui ti serve solo una funzione specifica di un pacchetto. Python ti permette di renderlo esplicito nel codice. Supponiamo di voler usare solo la funzione array del pacchetto NumPy. Invece di fare import numpy, puoi fare from numpy import array, così.

Questa volta puoi semplicemente chiamare la funzione array così, senza dover usare numpy punto.

Questa versione con from import per usare parti specifiche di un pacchetto può essere utile per scrivere meno codice, ma perdi anche un po’ di contesto.

## importa numpy

```yaml
type: FullSlide
key: b287cdae79
code_zoom: 70
```

`@part1`
```py
import numpy as np

fam = ["liz", 1.73, "emma", 1.68, 
	"mom", 1.71, "dad", 1.89]

...
```
```py
fam_ext = fam + ["me", 1.79]

...
```
```py
print(str(len(fam_ext)) + " elements in fam_ext")

...
```
```py
np_fam = np.array(fam_ext) # Clearly using NumPy
```{{1}}

`@script`
Da questo punto di vista, è preferibile la chiamata più standard import numpy: in questo caso, la tua chiamata diventa numpy.array, rendendo chiarissimo che stai lavorando con NumPy.
