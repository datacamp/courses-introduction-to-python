`@script`
Bunu yapmadan önce, paketi ya da paketin belirli bir modülünü içe aktarmalısın. Bunu import deyimiyle yapabilirsin.

Tüm numpy paketini içe aktarmak için, şöyle import numpy yazabilirsin.

NumPy’de sık kullanılan bir fonksiyon array’dir. Girdi olarak bir liste alır. Bu şekilde doğrudan array fonksiyonunu çağırırsan, hata alırsın.

numpy paketindeki array fonksiyonuna atıfta bulunmak için buna ihtiyacın var.

Bu sefer çalıştı. NumPy dizisi veri bilimi yapmak için çok kullanışlıdır, ama buna birazdan döneceğiz.

Sürekli bu numpy nokta önekini kullanmak yorucu olabilir, bu yüzden paketi içe aktarırken ona farklı bir adla da başvurabilirsin. Bunu import deyimini as ile genişleterek yapabilirsin, şöyle.

Artık NumPy’nin array fonksiyonunu kullanmak için numpy.array yerine np.array kullanacaksın.

Bazı durumlarda bir paketin yalnızca tek bir fonksiyonuna ihtiyaç duyarsın. Python, bunu kodunda açıkça belirtmene izin verir. Diyelim ki NumPy paketinden sadece array fonksiyonunu kullanmak istiyoruz. import numpy yapmak yerine, şöyle from numpy import array diyebilirsin.

Bu sefer, array fonksiyonunu bu şekilde doğrudan çağırabilirsin, burada numpy nokta kullanmana gerek yok.

Bir paketin belirli kısımlarını kullanmak için bu from import sürümü yazım miktarını azaltmak açısından yararlı olabilir, ancak biraz da bağlam kaybedersin.

Kodunu okuyan başka biri, bu array fonksiyonunun belirli bir NumPy fonksiyonu olduğunu unutmuş olabilir; fonksiyon çağrısından bu açık değildir.

import numpy as np

fam = ["liz", 1.73, "emma", 1.68, 
	"mom", 1.71, "dad", 1.89]

...
```
```py
fam_ext = fam + ["me", 1.79]

...
```
```py
print(str(len(fam_ext)) + " elements in fam_ext")

...
```
```py
np_fam = np.array(fam_ext) # NumPy açıkça kullanılıyor
```{{1}}

`@script`
Bu bakımdan, daha standart olan import numpy çağrısı tercih edilir: Bu durumda fonksiyon çağrın numpy.array olur ve NumPy ile çalıştığın çok net anlaşılır.
