```{{11}}

`@script`
Bevor du das machen kannst, musst du das Paket oder ein bestimmtes Modul des Pakets importieren. Nutze dazu die „import“-Anweisung.

Um das komplette NumPy-Paket zu importieren, kannst du „import NumPy“ wie folgt nutzen.

Eine häufig verwendete Funktion in NumPy ist „array“. Diese nimmt eine Liste als Eingabe entgegen. Wenn du die Array-Funktion einfach so aufrufst, wird ein Fehler ausgegeben.

Um auf die Array-Funktion aus dem NumPy-Paket zu verweisen, benötigst du Folgende Eingabe.

Diesmal klappt es. Das NumPy-Array ist super praktisch für Data Science, aber dazu später mehr.

Das NumPy-Präfix immer voranzustellen, kann ziemlich nervig werden. Du kannst das Paket also auch importieren und mit einem anderen Namen darauf verweisen. Das kannst du tun, indem du deine Import-Anweisung mit „as“ erweiterst, so wie hier.

Jetzt musst du anstatt „NumPy Punkt array“ lediglich „n-p Punkt array schreiben, um die Array-Funktion von NumPy zu nutzen.

Manchmal benötigst du nur eine bestimmte Funktion aus einem Paket. Mit Python kannst du das in deinem Code explizit angeben. Angenommen, wir wollen „nur“ die Array-Funktion aus dem NumPy-Paket verwenden. Anstatt „import NumPy“ zu verwenden, kannst du stattdessen „from NumPy import array“ verwenden, wie hier gezeigt.

Diesmal kannst du die Array-Funktion einfach so aufrufen, ohne „NumPy Punkt“ zu verwenden.

Das Importieren bestimmter Teile eines Pakets kann nützlich sein, um den Programmieraufwand zu reduzieren. Du verlierst dabei aber auch einen Teil des Kontexts.

## import numpy

```yaml
type: FullSlide
key: b287cdae79
code_zoom: 70
```

`@part1`
```py
import numpy as np

fam = ["liz", 1.73, "emma", 1.68, 
	"mom", 1.71, "dad", 1.89]

...
```
```py
fam_ext = fam + ["me", 1.79]

...
```
```py
print(str(len(fam_ext)) + " elements in fam_ext")

...
```
```py
np_fam = np.array(fam_ext) # Clearly using NumPy
```{{1}}

`@script`
In diesem Fall ist der normale „import NumPy“ Befehl zu bevorzugen: Damit ist dein Funktionsaufruf „NumPy Punkt array“, was deutlich macht, dass du mit NumPy arbeitest.
