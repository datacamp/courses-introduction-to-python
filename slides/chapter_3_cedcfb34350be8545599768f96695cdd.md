`@script`
Voordat je dat kunt doen, moet je het pakket importeren, of een specifieke module uit het pakket. Dat doe je met de import-instructie.

Om het volledige numpy-pakket te importeren, kun je import numpy doen, zo.

Een veelgebruikte functie in NumPy is array. Die neemt een lijst als input. Als je de array-functie zo aanroept, krijg je een foutmelding.

Om naar de array-functie uit het numpy-pakket te verwijzen, heb je dit nodig.

Deze keer werkt het. De NumPy-array is heel handig voor data science, maar daarover later meer.

Steeds dat voorvoegsel numpy punt gebruiken kan best vermoeiend worden, dus je kunt het pakket ook importeren en er met een andere naam naar verwijzen. Dat doe je door je import-instructie uit te breiden met as, zoals hier.

Nu gebruik je in plaats van numpy.array gewoon np.array om de array-functie van NumPy te gebruiken.

Er zijn situaties waarin je maar één specifieke functie uit een pakket nodig hebt. Python laat je dat expliciet maken in je code. Stel dat we alleen de array-functie uit het NumPy-pakket willen gebruiken. In plaats van import numpy te doen, kun je from numpy import array gebruiken, zoals hier.

Deze keer kun je de array-functie gewoon zo aanroepen, zonder numpy punt te gebruiken.

Deze from import-variant om specifieke delen van een pakket te gebruiken kan handig zijn om minder te hoeven typen, maar je verliest ook wat context.

fam_ext = fam + ["me", 1.79]

```
```py

```{{1}}

`@script`
