`@script`
Avant de pouvoir procéder, il est nécessaire d'importer le paquet ou un module spécifique du paquet. Vous pouvez le faire à l'aide de l’instruction import.

Pour importer l'intégralité du paquet numpy, vous pouvez utiliser la commande import numpy, comme ceci.

Une fonction couramment utilisée dans NumPy est array. Elle prend une liste en entrée. Le simple fait d'appeler la fonction array de cette manière générera une erreur.

Pour faire référence à la fonction array du paquet numpy, vous aurez besoin de ceci.

Cette fois, cela fonctionne. Le tableau NumPy est très utile pour la science des données, mais nous y reviendrons plus tard.

Utiliser systématiquement ce préfixe numpy dot peut devenir fastidieux. Vous pouvez donc importer le paquet et y faire référence sous un autre nom. Vous pouvez le faire en étendant votre instruction d'importation avec as, comme ceci.

Désormais, au lieu de numpy.array, vous devrez utiliser np.array pour utiliser la fonction de tableau de NumPy.

Dans certains cas, vous n'avez besoin que d'une seule fonction spécifique d'un paquet. Python vous permet de rendre cela explicite dans votre code. Supposons que nous souhaitions uniquement utiliser la fonction array du paquet NumPy. Au lieu d'utiliser « import numpy », vous pouvez utiliser « from numpy import array », comme ceci.

Cette fois-ci, vous pouvez simplement appeler la fonction array comme ceci, sans avoir besoin d'utiliser numpy dot ici.

Cette méthode consistant à importer une version afin d'utiliser des parties spécifiques d'un paquet peut être utile pour limiter la quantité de code, mais vous perdez également une partie du contexte.
