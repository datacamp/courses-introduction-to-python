```{{8}}

```py

```{{9}}
```py
array([1, 2, 3])
```{{10}}

```out
array([1, 2, 3])
```{{11}}

`@script`
Antes de poder hacerlo, debes importar el paquete o un módulo específico del paquete. Puedes hacerlo con la declaración import.

Para importar todo el paquete numpy, puedes hacer import numpy, así.

Una función muy utilizada en NumPy es array. Toma una lista como entrada. Al llamar a la función array de esta manera, se generará un error.

Para hacer referencia a la función array desde el paquete numpy, necesitarás esto.

Esta vez funciona. La matriz NumPy es muy útil para la ciencia de datos, pero hablaremos más sobre eso más adelante.

Usar este prefijo de numpy dot todo el tiempo puede resultar bastante tedioso, por lo que también puedes importar el paquete y hacer referencia a él con un nombre diferente. Una opción es ampliar la declaración import con as, de esta manera.

Ahora, en lugar de numpy.array, tendrás que usar np.array para utilizar la función array de NumPy.

Hay casos en los que solo necesitas una función específica de un paquete. Python te permite especificar esto en el código. Supongamos que solo queremos usar la función array del paquete NumPy. En lugar de hacer import numpy, puedes hacer from numpy import array. Así.

Esta vez, puedes llamar a la función array de esta manera, sin necesidad de usar numpy dot aquí.

Esto, en la versión de importación para utilizar partes específicas de un paquete, puede ser útil para limitar la cantidad de código, aunque también se pierde parte del contexto.

## import numpy

```yaml
type: FullSlide
key: b287cdae79
code_zoom: 70
```

`@part1`
```py
import numpy as np

fam = ["liz", 1.73, "emma", 1.68, 
	"mom", 1.71, "dad", 1.89]

...
```
```py
fam_ext = fam + ["me", 1.79]

...
```
```py
print(str(len(fam_ext)) + " elements in fam_ext")

...
```
```py
np_fam = np.array(fam_ext) # Clearly using NumPy
```{{1}}

`@script`
A tal fin, es preferible la llamada de import numpy más estándar: En este caso, la llamada a la función es numpy.array, lo que deja muy claro que estás trabajando con NumPy.
