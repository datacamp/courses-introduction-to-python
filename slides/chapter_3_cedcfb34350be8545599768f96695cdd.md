```py

```{{6}}
```py

```{{7}}

```out
array([1, 2, 3])
```{{8}}

```py
from numpy import array
```{{9}}
```py
array([1, 2, 3])
```{{10}}

```out
array([1, 2, 3])
```{{11}}

`@script`
Antes de fazer isso, devemos importar o pacote ou um módulo específico do pacote. Podemos fazer isso com a instrução import.

Para importar o pacote numpy inteiro, podemos fazer import numpy, assim.

Uma função muito usada no NumPy é array. Ela recebe uma lista como entrada. Apenas chamar a função array assim gera um erro.

Para fazer referência à função array do pacote numpy, precisamos disto.

Desta vez funcionou. O array do NumPy é superútil para ciência de dados, mas vamos falar mais disso depois.

Usar esse prefixo numpy ponto o tempo todo pode ser cansativo, então também podemos importar o pacote e chamá-lo de um nome diferente. Podemos fazer isso estendendo a instrução import com as, assim.

Agora, em vez de numpy ponto array, temos que usar np ponto array para utilizar a função array do NumPy.

Às vezes, precisamos somente de uma função específica de um pacote. O Python permite deixar isso bem claro no código. Suponha que a gente queira usar apenas a função array do pacote NumPy. Em vez de usar import numpy, podemos escrever from numpy import array, deste jeito.

Assim, basta chamar a função array desta maneira, sem precisar usar numpy ponto aqui.

Esta versão from import para usar partes específicas de um pacote pode ser útil para limitar a quantidade de código, mas também perdemos um pouco do contexto.

## import numpy

```yaml
type: FullSlide
key: b287cdae79

```

`@part1`
```py
import numpy as np

fam = ["liz", 1.73, "emma", 1.68, 
	"mom", 1.71, "dad", 1.89]

...
```
```py
fam_ext = fam + ["me", 1.79]

...
```
```py
print(str(len(fam_ext)) + " elements in fam_ext")

...
```
```py
np_fam = np.array(fam_ext) # Clearly using NumPy
```{{1}}

`@script`
Nesse sentido, é preferível usar a chamada import numpy mais comum. Nesse caso, a chamada da função é numpy ponto array, deixando bem claro que você está trabalhando com o NumPy.
