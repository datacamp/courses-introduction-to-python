```{{11}}

`@script`
Trước khi dùng, ta cần import package, hoặc một module cụ thể của package bằng cách sử dụng câu lệnh import.


Để import toàn bộ package numpy, ta viết lệnh import như thế này. 

Một hàm thường xuyên dùng trong NumPy là array. Hàm này nhận một list làm đầu vào. Nếu ta chỉ gọi hàm array như thế này, Python sẽ báo lỗi.

Để gọi hàm array từ package numpy, ta cần viết như sau. 


Lúc này code sẽ chạy đúng. NumPy array rất hữu ích cho khoa học dữ liệu, nhưng chúng ta sẽ nói kĩ hơn sau. 

Cứ phải gõ numpy chấm suốt cũng khá bất tiện, nên ta có thể import package và đặt cho nó một tên ngắn hơn. Chúng ta có thể làm bằng cách thêm as vào câu lệnh import, như thế này.


Giờ, thay vì numpy.array, bạn chỉ cần gõ np.array để gọi hàm array của NumPy.

Trong một số trường hợp ta chỉ cần một hàm cụ thể từ một package. Python cho phép ta viết rõ điều này trong code. Giả sử ta chỉ cần dùng hàm array từ package NumPy. Thay vì viết import numpy, ta viết from numpy import array như thế này.


Lúc này, ta có thể gọi trực tiếp hàm array như thế này mà không cần dùng numpy chấm nữa.

Cách viết from import để dùng các phần cụ thể từ một package giúp ta phải code ít hơn, nhưng nó cũng làm giảm bớt thông tin.
