`@script`
Les fonctions ne sont pas entièrement nouvelles pour vous : vous les avez déjà utilisées. type, par exemple, est une fonction qui renvoie le type d'une valeur. Cependant, qu'est-ce qu'une fonction ? En termes simples, une fonction est un morceau de code réutilisable, destiné à résoudre une tâche particulière. Vous pouvez appeler des fonctions au lieu d'écrire vous-même du code. Un exemple permettra peut-être de clarifier les choses.

    Otherwise the return value has the same type as the number. ndigits may be negative.

`@script`
Une autre de ces fonctions intégrées est round. Elle nécessite deux entrées : premièrement, le nombre à arrondir, et deuxièmement, la précision avec laquelle il doit être arrondi, c'est-à-dire le nombre de chiffres après la virgule décimale que vous souhaitez conserver. Supposons que vous souhaitiez arrondir 1 virgule 68 à une décimale. La première entrée est 1 virgule 68, la deuxième entrée est 1. Vous séparez les entrées par une virgule.

Presque, mais ce n'est pas tout. Il est tout à fait possible d'appeler la fonction round avec une seule entrée, comme ceci. Cette fois, Python a détecté que vous n'aviez pas spécifié la deuxième entrée et a automatiquement choisi d'arrondir le nombre à l'entier le plus proche.

Pour comprendre pourquoi ces deux approches marchent, veuillez consulter la documentation. Vous pouvez réaliser cela à l'aide d'une autre fonction, help, comme suit.

Il semble que round nécessite deux entrées.
