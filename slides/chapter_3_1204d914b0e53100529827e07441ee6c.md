Help on built-in function round in module builtins:

round(number, ndigits=None)
    Round a number to a given precision in decimal digits.
    
    The return value is an integer if ndigits is omitted or None. 
    Otherwise the return value has the same type as the number. ndigits may be negative. 
```{{3}}

`@script`
내장 함수 중에 round도 있어요. 이 함수는 입력을 2개 받습니다. 첫째, 반올림할 숫자, 둘째, 반올림 정밀도인데, 소수점 아래 몇 자리까지 남길지를 의미해요. 예를 들어 1.68을 소수점 첫째 자리까지 반올림하고 싶다면 첫 번째 입력은 1.68, 두 번째 입력은 1이 됩니다. 입력은 쉼표로 구분해요.

하지만 더 있어요. round를 이렇게 입력 1개만으로 호출해도 됩니다. 이번에는 두 번째 입력을 지정하지 않았다는 걸 Python이 알아채고, 가장 가까운 정수로 반올림하도록 자동으로 처리해요.

두 방식이 모두 가능한 이유를 이해하려면, 문서를 열어 보면 돼요. 또 다른 함수인 help를 이렇게 사용하면 됩니다.

문서를 보면 round는 입력을 2개 받는다고 하네요.
