```{{3}}

```out
Help on built-in function round in module builtins:

round(number, ndigits=None)
    Round a number to a given precision in decimal digits.
    
    The return value is an integer if ndigits is omitted or None. 
    Otherwise the return value has the same type as the number. ndigits may be negative. 
```{{3}}

`@script`
Otra de estas funciones integradas es round. Se necesitan dos datos: primero, el número que deseas redondear y, segundo, la precisión con la que deseas redondearlo (es decir, cuántos dígitos después del punto decimal deseas mantener). Supongamos que quieres redondear 1,68 a un decimal. La primera entrada es 1,68 y la segunda entrada es 1. Separa las entradas con una coma.

Pero hay más. Es perfectamente posible llamar a la función round con una sola entrada, como se muestra a continuación. Esta vez, Python ha detectado que no has especificado la segunda entrada y ha optado automáticamente por redondear el número al entero más cercano.

Para entender por qué funcionan ambos enfoques, abre la documentación. Puedes hacerlo con otra función, help, de esta manera.

Parece que round toma dos entradas.
