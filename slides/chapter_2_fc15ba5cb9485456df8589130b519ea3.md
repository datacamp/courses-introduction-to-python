```{{5}}

![Sintaxe de fatiamento ou slicing de listas em Python, mostrando que o valor inicial é incluído no subconjunto, enquanto o valor final é excluído.](https://assets.datacamp.com/img/translations/pt-BR/production/repositories/288/datasets/83dd2f807be0d4d08a187935eed11667c18fcfe3/slicing-syntax.png = 40){{3}}

`@script`
que permite selecionar vários elementos de uma lista, criando assim uma nova lista. Você pode fazer isso especificando um intervalo, usando dois pontos. Vamos dar mais uma olhada na lista e testar este código.

Dá para adivinhar qual será o resultado? Uma lista com o float um ponto sessenta e oito, a string “mom” e o float um ponto setenta e um, que correspondem ao quarto, o quinto e o sexto elementos da lista, talvez? Vamos ver qual é o resultado.

Parece que somente os elementos com índice três e quatro são retornados. O elemento com índice cinco não foi incluído. Em geral, esta é a sintaxe: o índice que você especifica antes dos dois pontos, ou seja, onde a fatia começa, é incluído, enquanto o índice que você coloca depois dos dois pontos, ou seja, onde a fatia termina, não é incluído.

Pensando nisso, dá para saber o que esta chamada vai retornar?

Você deve ter adivinhado que esta chamada mostra uma lista com três elementos, que correspondem aos elementos com os índices um, dois e três da lista fam.

Você também pode simplesmente omitir o índice antes ou depois dos dois pontos.
