```py

```{{1}}

```out

```{{2}}

```py

```{{4}}

```out
[1.73, 'emma', 1.68]
```{{5}}

![La sintaxis de uso de slices para listas de Python, que muestra que el valor inicial está incluido en el subconjunto y que el valor final está excluido.](https://assets.datacamp.com/img/translations/es-ES/production/repositories/288/datasets/83dd2f807be0d4d08a187935eed11667c18fcfe3/slicing-syntax.png = 40){{3}}

`@script`
que te permite seleccionar varios elementos de una lista para crear otra nueva. Para ello, puedes especificar un rango utilizando dos puntos. Echemos otro vistazo a la lista y, a continuación, probemos este fragmento de código.

¿Puedes adivinar qué te devolverá? ¿Una lista con el número flotante 1,68, la cadena mom y el número flotante 1,71, correspondientes al cuarto, quinto y sexto elemento de la lista, tal vez? Veamos cuál es el resultado.

Aparentemente, solo se devuelven los elementos con índice 3 y 4. El elemento con índice 5 no está incluido. Por lo general, esta es la sintaxis: el índice que especificas antes de los dos puntos (es decir, donde comienza el slice) se incluye, mientras que el índice que especificas después de los dos puntos (es decir, donde termina el slice) no se incluye.

Teniendo esto en cuenta, ¿puedes decir qué devolverá esta llamada?

Probablemente hayas adivinado correctamente que esta llamada te da una lista con tres elementos, correspondientes a los elementos con índice 1, 2 y 3 de la lista fam.

También puedes omitir el índice antes o después de los dos puntos.
