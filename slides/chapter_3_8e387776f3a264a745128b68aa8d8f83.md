`@script`
fam というリストの中で、文字列 "mom" の位置を知りたいとします。fam は list 型の Python オブジェクトで、index という名前のメソッドを持っています。メソッドを呼び出すには、このようにドット記法を使います。入力は、位置を調べたい要素である文字列 "mom" だけです。

Python は 4 を返します。これは確かに文字列 "mom" のインデックスですね。ここでは fam リスト「に対して」index メソッドを呼び出し、出力は 4 でした。同様に、count メソッドを fam リストに対して使うと、1.73 が何回出現するかを数えられます。

Python は 1 を返します。1.73 メートルなのは liz だけなので、納得できますね。
