`@script`
Stel, je wilt de index van de string "mom" in de fam-lijst ophalen. fam is een Python-object van het type list en heeft een methode genaamd index. Om de methode aan te roepen gebruik je de puntnotatie. De enige input is de string "mom", het element waarvan je de index wilt weten.

Python geeft vier terug, wat inderdaad de index is van de string "mom". Ik riep de methode index hier "op" de fam-lijst aan, en de uitvoer was vier. Op dezelfde manier kan ik de methode count op de fam-lijst gebruiken om te tellen hoe vaak één-komma-drieën-zeventig in de lijst voorkomt.

Python geeft mij één. Dat is logisch, want alleen liz is één-komma-drieën-zeventig meter lang.
