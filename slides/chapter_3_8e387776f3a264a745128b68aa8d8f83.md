`@script`
Angenommen, du willst den Index der Zeichenfolge „mom” in der Liste „fam” herausfinden. „fam” ist ein Python Objekt vom Typ „list” und hat eine Methode namens „index”. Um die Methode aufzurufen, benutzt du die Punktnotation, so wie hier gezeigt. Der einzige Input ist die Zeichenfolge „mom“, also das Element, für das du den Index ermitteln willst.

Python gibt „vier“ zurück, was tatsächlich der Index der Zeichenfolge „mom” ist. Ich habe die Indexmethode „auf“ der „fam“-Liste aufgerufen und die Ausgabe war „vier“. Genauso kann ich die Methode „count“ auf die Liste „fam“ anwenden, um zu zählen, wie oft „eins-Komma-sieben-drei“ in der Liste vorkommt.

Python gibt mir „Eins“ zurück, was Sinn ergibt, weil nur „liz“ „eins-Komma-sieben-drei“ Meter groß ist.
