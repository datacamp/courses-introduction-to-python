`@part1`
```py
fam = ["liz", 1.73, "emma", 1.68, "mom", 1.71, "dad", 1.89]

```

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.89]
```

```py
fam[7] = 1.86
fam
```{{1}}

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.86]
```{{1}}

```py
fam[0:2] = ["lisa", 1.74]
fam
```{{2}}

```out
['lisa', 1.74, 'emma', 1.68, 'mom', 1.71, 'dad', 1.86]
```{{2}}

`@script`
Thay đổi phần tử trong list khá đơn giản. Bạn vẫn dùng dấu ngoặc vuông như khi truy cập cắt list, rồi gán giá trị mới bằng dấu bằng. Giả sử sau khi xem lại fam, bạn nhận ra chiều cao của bố không còn đúng nữa vì bố đã thấp đi do tuổi già. Thay vì 1,89 mét, giờ phải là 1,86 mét. Phần tử này nằm ở index 7, để thay đổi nó, bạn dùng dòng code sau.


Nếu giờ bạn kiểm tra fam, bạn sẽ thấy giá trị đã được cập nhật.

Bạn còn có thể thay đổi cả một đoạn của list cùng lúc. Để thay đổi các phần tử "liz" và 1,73, bạn truy cập hai phần tử đầu bằng với 0:2, rồi gán một list mới cho nó.

Bạn còn nhớ toán tử cộng hoạt động khác nhau giữa string và integer không?
