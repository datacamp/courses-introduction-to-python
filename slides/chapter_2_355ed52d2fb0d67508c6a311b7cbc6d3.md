## Mengubah elemen daftar

```yaml
type: FullSlide
key: c1d58a3c4c
code_zoom: 64
```

`@part1`
```py
fam = ["liz", 1.73, "emma", 1.68, "mom", 1.71, "dad", 1.89]
fam
```

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.89]
```

```py
fam[7] = 1.86
fam
```{{1}}

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.86]
```{{1}}

```py
fam[0:2] = ["lisa", 1.74]
fam
```{{2}}

```out
['lisa', 1.74, 'emma', 1.68, 'mom', 1.71, 'dad', 1.86]
```{{2}}

`@script`
Mengubah elemen daftar cukup mudah. Anda menggunakan tanda kurung siku yang sama seperti saat subsetting daftar, lalu menetapkan elemen baru dengan tanda sama dengan. Misalkan setelah melihat keluarga lagi, Anda sadar tinggi badan ayah Anda sudah tidak mutakhir, karena beliau menyusut seiring usia. Tidak lagi 1.89 meter, sekarang 1.86 meter. Untuk mengubah elemen daftar ini, yang berada di indeks 7, Anda bisa gunakan baris kode ini.

Jika sekarang Anda memeriksa keluarga, Anda akan melihat nilainya sudah diperbarui.

Anda bahkan bisa mengubah satu irisan daftar sekaligus. Untuk mengganti elemen "liz" dan 1.73, Anda mengakses dua elemen pertama dengan 0:2, lalu tetapkan daftar baru ke sana.

Apakah Anda masih ingat bagaimana operator plus berbeda untuk string dan integer?

```out

```{{1}}

```py
fam_ext = fam + ["me", 1.79]
```{{2}}
```py
del fam[2]
```{{3}}
```py
fam
```{{4}}

```out
['lisa', 1.74, 1.68, 'mom', 1.71, 'dad', 1.86]
```{{4}}

`@script`
Nah, untuk daftar pun berbeda lagi. Jika Anda memakai tanda plus pada dua daftar, Python akan menempelkan isi keduanya menjadi satu daftar. Misalkan Anda ingin menambahkan nama dan tinggi badan Anda ke daftar tinggi fam. Ini akan menyelesaikannya.

Tentu saja, Anda juga bisa menyimpan daftar baru ini ke dalam sebuah variabel, misalnya keluarga_ext.

Terakhir, menghapus elemen dari daftar juga cukup mudah, Anda harus menggunakan del di sini. Ambil baris ini, misalnya, yang menghapus elemen dengan indeks 2, yaitu "emma", dari daftar.

Jika sekarang Anda memeriksa keluarga, Anda akan melihat string "emma" sudah hilang. Karena Anda menghapus sebuah indeks, semua elemen setelah "emma" bergeser satu indeks. Jika Anda menjalankan baris yang sama lagi, Anda kembali menghapus elemen di indeks 2, yaitu tinggi emma, sekarang 1.68 meter.
