`@part1`
```py
fam = ["liz", 1.73, "emma", 1.68, "mom", 1.71, "dad", 1.89]
fam
```

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.89]
```

```py
fam[7] = 1.86
fam
```{{1}}

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.86]
```{{1}}

```py
fam[0:2] = ["lisa", 1.74]
fam
```{{2}}

```out
['lisa', 1.74, 'emma', 1.68, 'mom', 1.71, 'dad', 1.86]
```{{2}}

`@script`
リストの要素を変えるのはとても簡単です。リストの抽出で使ったのと同じ角かっこを使い、イコールで新しい値を代入します。たとえば fam を見直して、お父さんの身長が年齢とともに低くなっていて最新ではないと気づいたとします。1.89 メートルではなく、1.86 メートルにすべきですね。この要素はインデックス 7 にあるので、次のコードで変更できます。

今 fam を確認すると、値が更新されているのがわかります。

リストのスライス全体を一度に変えることもできます。"liz" と 1.73 を変えたいなら、0:2 で最初の 2 要素にアクセスし、新しいリストを代入します。

文字列と整数でプラス演算子の振る舞いが違っていたのを覚えていますか？

## 要素の追加と削除

```out
['lisa', 1.74,'emma', 1.68, 'mom', 1.71, 'dad', 1.86, 'me', 1.79]
```{{1}}

```py
fam_ext = fam + ["me", 1.79]
```{{2}}
```py
del fam[2]
```{{3}}
```py
fam
```{{4}}

```out
['lisa', 1.74, 1.68, 'mom', 1.71, 'dad', 1.86]
```{{4}}

`@script`
実は、リストでもまた違います。2 つのリストにプラス記号を使うと、Python はその中身を 1 つのリストとして単純に連結します。自分の名前と身長を fam のリストに追加したいとしましょう。これでうまくいきます。

もちろん、この新しいリストを変数に保存することもできます。たとえば fam_ext です。

最後に、リストから要素を削除するのもとても簡単で、ここでは del を使います。たとえばこの行は、インデックス 2、つまり "emma" をリストから削除します。

今 fam を確認すると、"emma" という文字列がなくなっているのがわかります。1 つインデックスを削除したので、"emma" の後ろにあった要素はすべてインデックスが 1 つずつ前に詰まりました。同じ行をもう一度実行すると、再びインデックス 2 の要素、つまり emma の身長 1.68 メートルが削除されます。
