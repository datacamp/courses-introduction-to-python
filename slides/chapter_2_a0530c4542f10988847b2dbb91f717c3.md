fam = ["liz", 1.73, "emma", 1.68, "mom", 1.71, "dad", 1.89]
```
```py
fam
```

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.89]
```

```py
fam2 = [["liz", 1.73],
		["emma", 1.68],
		["mom", 1.71],
		["dad", 1.89]]
```{{1}}
```py
fam2
```{{2}}

```out
[['liz', 1.73], ['emma', 1.68], ['mom', 1.71], ['dad', 1.89]]
```{{2}}

`@script`
Misalnya, Anda ingin menambahkan nama saudara perempuan dan orang tua Anda ke dalam daftar, agar Anda tahu tinggi mana milik siapa. Anda bisa memasukkan beberapa string tanpa masalah.

Namun itu bukan semuanya. Saya baru saja memberi tahu bahwa daftar juga bisa berisi daftar di dalamnya. Alih-alih menaruh string di antara angka-angka, Anda dapat membuat subdaftar kecil untuk setiap anggota keluarga. Satu untuk liz, satu untuk emma, dan seterusnya. Sekarang, Anda bisa memberi tahu Python bahwa subdaftar ini adalah elemen dari daftar lain, yang saya beri nama keluarga2: daftar-daftar kecil tersebut dibungkus dengan kurung siku dan dipisahkan dengan koma. Jika sekarang Anda mencetak keluarga2, Anda akan melihat bahwa kita punya daftar berisi daftar. Daftar utama berisi 4 subdaftar.

Di sini kita berhadapan dengan tipe Python baru, selain string, boolean, integer, dan float yang sudah Anda kenal:
