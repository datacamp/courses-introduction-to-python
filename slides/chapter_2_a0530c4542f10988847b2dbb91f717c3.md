fam = ["liz", 1.73, "emma", 1.68, "mom", 1.71, "dad", 1.89]

```py

```

```out
['liz', 1.73, 'emma', 1.68, 'mom', 1.71, 'dad', 1.89]
```

```py
fam2 = [["liz", 1.73],
		["emma", 1.68],
		["mom", 1.71],
		["dad", 1.89]]
```{{1}}
```py
fam2
```{{2}}

```out
[['liz', 1.73], ['emma', 1.68], ['mom', 1.71], ['dad', 1.89]]
```{{2}}

`@script`
Por ejemplo, supongamos que quieres añadir los nombres de tus hermanas y padres a la lista para saber a quién corresponde cada altura. Puedes añadir algunas cadenas sin problemas.

Pero eso no es todo. Acabo de decirte que las listas también pueden contener otras listas. En lugar de poner las cadenas entre los números, puedes crear pequeñas sublistas para cada miembro de la familia. Uno para Liz, otro para Emma, y así sucesivamente. Ahora puedes decirle a Python que estas sublistas son los elementos de otra lista, que he llamado fam2: las pequeñas listas se envuelven entre corchetes y se separan con comas. Si ahora imprimes fam2, verás que tenemos una lista de listas. La lista principal contiene 4 sublistas.

Aquí nos encontramos con un nuevo tipo de Python, además de las cadenas, los booleanos, los enteros y los flotantes que ya conoces:
