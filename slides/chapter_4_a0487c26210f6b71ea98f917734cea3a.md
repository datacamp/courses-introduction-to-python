np.array([1.0, "is", True])

```out

- Array NumPy: hanya berisi satu jenis{{1}}

`@script`
Namun Anda tetap harus memperhatikan. Pertama, NumPy bisa melakukan semua ini dengan mudah karena berasumsi bahwa NumPy array Anda hanya boleh berisi nilai dengan satu tipe saja. Entah berupa array float, atau array boolean, dan seterusnya. Jika Anda mencoba membuat array dengan tipe berbeda, seperti ini misalnya, NumPy array yang dihasilkan akan berisi satu tipe saja, string dalam kasus ini. Boolean dan float keduanya dikonversi menjadi string.

Kedua, Anda perlu tahu bahwa NumPy array hanyalah jenis tipe Python baru, seperti tipe float, string, dan daftar sebelumnya. Artinya, ia memiliki metode sendiri, yang bisa berperilaku berbeda dari yang Anda harapkan.

## Subsetting NumPy

```yaml
type: FullSlide
key: c1f3774f83
code_zoom: 71
```

`@part1`
```py
bmi
```{{1}}

```out
array([21.85171573, 20.97505669, 21.75028214, 24.7473475 , 21.44127836])
```{{1}}

```py
bmi[1]
```{{2}}

```out
20.975
```{{2}}

```py
bmi > 23
```{{3}}

```out
array([False, False, False,  True, False])
```{{3}}

```py
bmi[bmi > 23]
```{{4}}

```out
array([24.7473475])
```{{4}}

`@script`
Anda dapat bekerja dengan NumPy array hampir sama seperti dengan daftar Python biasa. Ketika Anda ingin mengambil elemen dari array, misalnya, Anda bisa menggunakan tanda kurung siku. Misalkan Anda ingin mendapatkan bmi untuk orang kedua, jadi pada indeks 1. Ini akan berhasil.

Khusus untuk NumPy, ada cara lain untuk melakukan subsetting daftar, menggunakan array boolean. Misalnya Anda ingin mendapatkan semua nilai BMI di array bmi yang di atas 23. Langkah pertama adalah menggunakan tanda lebih besar, seperti ini:

Hasilnya adalah NumPy array yang berisi boolean: True jika bmi yang bersangkutan di atas 23, False jika di bawah. Berikutnya, Anda bisa menggunakan array boolean ini di dalam kurung siku untuk melakukan subsetting. Hanya elemen di bmi yang di atas 23, jadi yang nilai boolean bersesuaian bernilai True, yang dipilih. Hanya ada satu BMI yang di atas 23, jadi kita mendapatkan NumPy array dengan satu nilai, BMI tersebut.

Menggunakan hasil perbandingan untuk menyeleksi data Anda adalah cara yang sangat umum untuk mendapatkan wawasan yang mengejutkan.
