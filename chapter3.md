O Python oferece uma série de funções integradas para facilitar a vida de um cientista de dados. Você já conhece duas dessas funções: `print()` e `type()`. Há também funções como `str()`, `int()`, `bool()` e `float()` para alternar entre os tipos de dados. Você pode saber mais sobre eles [aqui.](https://docs.python.org/3/library/functions.html) Essas também são funções integradas.

Chamar uma função é fácil. Para obter o tipo de `3.0` e armazenar a saída como uma nova variável, `result`, você pode usar o seguinte:

```

```

`@hint`
- Chame a função `type()` da seguinte forma: `type(var1)`.
- Ligue para `print()` como você fez tantas vezes antes. Basta colocar a variável que deseja imprimir entre parênteses.
- `int(x)` converte `x` em um número inteiro.

Há várias maneiras de importar pacotes e módulos no Python. Dependendo da chamada de importação, deve ser usado um código Python diferente.

```
my_inv([[1,2], [3,4]])
```

Qual instrução `import` é necessária para executar o código acima sem erros?

`@possible_answers`
- `import scipy`
- `import scipy.linalg`
- `from scipy.linalg import my_inv`
- `from scipy.linalg import inv as my_inv`
